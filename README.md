# XiaoYao_快速跳转
在打开或保存对话框中，快速跳转到当前 资源管理器/TC/DO/XY/Q-Dir中打开的文件夹路径

Ps：由于本人主用DO文件管理器，所有还适配了do收藏夹。

有任何问题或者建议，欢迎前往 Github 提交 issue。也可加qq群交流：246308937

# 使用说明

在 打开或保存对话框 窗口界面中：

**快捷键 "Ctrl + G" 可呼出路径菜单**

**按快捷键 "Ctrl + F" 可一键跳转到当前活动路径**

**按快捷键 "Ctrl + D" 可呼出常驻跟随窗口**

可在“设置”里修改呼出热键，也可设置全局呼出。
